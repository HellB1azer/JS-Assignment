DataTypes in JavaScript.

There are different DataTypes in JavaScript.
Number DataType consists of integer, floating point number, etc.
Boolean DataType displays True or False statement.
When variable is not defined a specific value it displays undefined.
A null value means that there is no value. It is not equivalent to an empty string ("") or 0, it is simply nothing.
Object DataType is used to store collection of data.
Array DataType is used to store two or more values of a single variable.
